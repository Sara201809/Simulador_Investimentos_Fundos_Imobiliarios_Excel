# Simulador de Investimentos em Fundos Imobiliários

## 🎯 Objetivo
Este projeto foi desenvolvido como parte do bootcamp da DIO. O objetivo é utilizar uma ferramenta de planilha (Excel) para simular e analisar investimentos em Fundos Imobiliários (FIIs), auxiliando no controle financeiro pessoal.

## 🛠️ Tecnologias Utilizadas
- Excel (ou ferramenta de planilhas utilizada)
- Git e GitHub (Versionamento)

## 📝 Como acessar
1. O arquivo da planilha está disponível neste repositório.
2. Você pode baixar o arquivo `.csv` ou `.xlsx` para visualizar a estrutura da simulação.
